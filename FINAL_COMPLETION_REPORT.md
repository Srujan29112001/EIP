## 🚀 HOW TO RUN & DEPLOY

### Quick Start (Local Development)

```bash
# 1. Clone and setup
git clone <repo-url>
cd EIP
cp .env.example .env
# Edit .env with your API keys

# 2. Install dependencies
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt

# 3. Start services
docker-compose up -d postgres mongodb neo4j redis kafka

# 4. Initialize databases
python scripts/init_db.py

# 5. Start backend
cd backend
uvicorn app.main:app --reload

# 6. Start frontend (new terminal)
cd frontend
streamlit run app.py
```

**Access:**
- Frontend: http://localhost:8501
- Backend API: http://localhost:8000/docs

### Docker Deployment

```bash
# Build and start all services
docker-compose up -d

# View logs
docker-compose logs -f backend frontend

# Access dashboard
open http://localhost:8501
```

### Kubernetes Production

```bash
# Deploy to production cluster
kubectl apply -f infrastructure/k8s/ -n eip-prod

# Check status
kubectl get pods -n eip-prod

# Access application
kubectl get ingress -n eip-prod
```

**Full deployment guide:** See `BUILD_RUN_DEPLOY_GUIDE.md`
