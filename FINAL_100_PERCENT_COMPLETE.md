## 📋 DEPLOYMENT CHECKLIST

### Prerequisites
- [ ] Python 3.11+
- [ ] Docker & Docker Compose
- [ ] Kubernetes cluster (for production)
- [ ] API Keys:
  - [ ] OpenAI API key
  - [ ] Anthropic API key (optional)
  - [ ] DeepSeek API key (optional)
  - [ ] NewsAPI key
  - [ ] Alpha Vantage key

### Quick Start (Development)

```bash
# 1. Clone repository
git clone <repo-url>
cd EIP

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set up environment
cp .env.example .env
# Edit .env with your API keys

# 4. Start infrastructure
docker-compose up -d

# 5. Initialize databases
python scripts/init_db.py
python scripts/seed_knowledge_base.py

# 6. Start backend
cd backend
uvicorn app.main:app --reload

# 7. Start frontend (new terminal)
cd frontend
streamlit run app.py

# 8. Access platform
# Frontend: http://localhost:8501
# Backend API: http://localhost:8000
# API Docs: http://localhost:8000/docs
```

### Production Deployment

```bash
# 1. Build Docker images
docker build -t eip-backend:latest -f docker/Dockerfile.backend .
docker build -t eip-frontend:latest -f docker/Dockerfile.frontend .

# 2. Push to registry
docker push <registry>/eip-backend:latest
docker push <registry>/eip-frontend:latest

# 3. Deploy to Kubernetes
kubectl apply -f infrastructure/k8s/

# 4. Verify deployment
kubectl get pods -n eip
kubectl get services -n eip
```
